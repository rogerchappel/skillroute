# Support

`skillroute` is an early local-first CLI. The supported surface is the current `main` branch, the documented `skillroute plan` command, and the exported route planner helpers.

## Before Filing an Issue

Run:

```sh
npm run release:check
```

Include:

- The command you ran.
- A minimal redacted catalog and task prompt.
- The expected route and the route that was produced.
- Your Node.js version.

Please redact private skill names, proprietary prompts, credentials, and approval-policy details before sharing examples publicly.
